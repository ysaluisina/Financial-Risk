# Financial-Risk (Power BI)

Este dataset se obtuvo desde el sitio web de www.kaggle.com, y contiene información acerca del riesgo financiero teniendo en cuenta los perfiles, edad, y trabajo, entre otras cosas, de las personas que han solicitado algún tipo de crédito. Además, incluye datos demográficos, como es el país de los solicitantes. Abarca el período de años desde el 2009 hasta el 2017. 

## Objetivo

Analizar de manera estratégica la información que reflejan las tablas con el fin de reducir el riesgo financiero en un 20 % durante los próximos 6 meses, conociendo y teniendo en cuenta las características a evaluar de los solicitantes que aspiran a algún préstamos crediticios, para, posteriormente, aceptar o rechazar los préstamos solicitados.
